# 赞助支持

SmartAdmin 采用 Apache-2.0 协议开放源码。项目长期维护需要社区赞助、会员支持和商业服务收入来覆盖开发、文档、答疑和生态建设成本。

## 支持方式

| 方式 | 说明 |
|------|------|
| 项目赞助 | 通过当前收款二维码支持作者持续维护 |
| 会员支持 | 加入会员，获得会员权益和会员群交流资格 |
| 有偿服务 | 需要定制开发、疑难排查或专项技术支持时，可通过微信进一步沟通 |
