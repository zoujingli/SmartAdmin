## 再分发要求

- 保留 Apache License 2.0 协议正文。
- 保留源码中的版权、许可证和归属声明。
- 修改后再分发时，在被修改文件或发布说明中标记修改事实。
- 随分发包保留 `NOTICE` 中适用的归属信息。
- 第三方组件继续遵守各自原始许可证。

## 第三方组件

项目使用 Hyperf、Vue Vben Admin 和其他 Composer、pnpm 依赖。第三方组件的许可证不会因为 SmartAdmin 使用 Apache-2.0 而改变；发布衍生版本时，需要同时遵守这些组件的原始许可证。
