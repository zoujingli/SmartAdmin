# 更新日志

## 版本标记规范

- 文档版本建议采用 `v主版本.次版本.修订号`，例如 `v1.2.0`
- 纯文档修订可只提升修订号（例如 `v1.2.1`）
- 涉及结构级重建或章节大迁移，至少提升次版本
- 每条记录建议包含：日期、变更主题、影响范围、维护人（可选）
