## 样式基线

当前 docs 参考 DevAPI 工作台的阅读体验，统一采用：

| 项目 | 当前实现 |
|---|---|
| 默认入口 | 站点根路径直接展示 `快速开始/README.md` 正文 |
| 顶部导航 | `devapi-module-nav.js` 从侧边栏一级分组生成模块导航，并支持下载当前 Markdown |
| 侧边栏 | 中文业务分组 + 可折叠树形目录 + 展开/收起工具 + 滚动位置记忆 |
| 视觉资源 | `assets/devapi-docs.css`、`devapi-scale.js`、`devapi-module-nav.js`、`devapi-sidebar-state.js`、`devapi-mermaid.js` |
| 在线调试 | `assets/api-tester.css`、`api-tester.js` |
| 翻页 | `docsify-pagination` 提供上一章节/下一章节卡片 |

调整样式、脚本或导航结构时，必须同步 [文档系统部署](./文档系统部署.md) 和 `xadmin:docs:check` 校验项。

## 维护原则

- 代码变更先落地，再同步文档
- 规范变化更新对应主题文档，接口变化同步接口参考，版本变化更新日志
- 同一主题只保留一个主文档，避免重复与冲突
- 补充专题必须服务主线文档，避免形成第二套口径
