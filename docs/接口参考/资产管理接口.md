# 资产管理接口

> **会员授权 / 不随开源版源码分发：** 本页为会员授权插件文档，公开文档站展示接口与使用说明，但开源仓不包含该插件源码或 ZIP，安装使用需会员授权或内部交付。

Asset 插件是会员专属应用，不随 SmartAdmin 开源版开放分发；仅会员授权环境可安装和使用。

Asset 插件提供固定资产台账、一物一码、员工领退调维废申请、盘点、钉钉组织同步、工作通知和钉钉 OA 审批同步。后台接口使用 System Token 和 `asset.*` 权限码；员工端接口使用 `Plugin\Asset\Model\AssetEmployee` Token，员工身份只允许通过钉钉免登或已绑定员工密码登录获得。

## 员工端认证与个人入口

| 方法 | 路径 | 认证 | 说明 |
|---|---|---|---|
| GET | `/asset/auth/password-crypto` | 公开 | 获取员工密码登录的 RSA-OAEP 公钥与一次性 nonce。 |
| GET | `/asset/auth/dingtalk-config` | 公开 | 获取钉钉免登所需 CorpId，不返回服务端密钥。 |
| POST | `/asset/auth/dingtalk-login` | 公开 | 使用钉钉 auth code 换取 Asset 员工 Token。 |
| POST | `/asset/auth/refresh` | Asset Token | 刷新员工端 Token。 |
| POST | `/asset/auth/profile` | Asset Token | 获取当前员工资料。 |
| PUT | `/asset/auth/profile` | Asset Token | 更新当前员工资料。 |
| PUT | `/asset/auth/preferences` | Asset Token | 保存员工端界面偏好。 |
| PUT | `/asset/auth/password` | Asset Token | 修改员工登录密码。 |
| GET | `/asset/auth/codes` | Asset Token | 获取员工端权限码。 |
| POST | `/asset/auth/logout` | Asset Token | 登出员工端。 |
| GET | `/asset/profile` | Asset Token | 获取员工端首页资料。 |
| GET | `/asset/my-assets` | Asset Token | 查询当前员工名下资产。 |
| GET | `/asset/applications` | Asset Token | 查询当前员工资产申请。 |
| POST | `/asset/application/{type}` | Asset Token | 提交领用、退还、调拨、维修或报废申请。 |
| GET | `/asset/inventories` | Asset Token | 查询当前员工盘点任务。 |

## 业务边界

- Asset 员工端只使用 `AssetEmployee` Token，不复用 System 用户 Token。
- 钉钉组织同步只写入 Asset 自有员工快照和部门绑定，不创建或更新 System 用户。
- 领用、退还、调拨、维修、报废、盘点差异确认等动作必须落入资产操作日志，便于追溯责任人、资产状态和位置变化。
- Asset 插件是会员专属插件，公开 `SmartAdmin` 导出会移除源码、文档和 Composer 依赖。
