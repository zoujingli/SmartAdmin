# 积分管理接口

> **会员授权 / 不随开源版源码分发：** 本页为会员授权插件文档，公开文档站展示接口与使用说明，但开源仓不包含该插件源码或 ZIP，安装使用需会员授权或内部交付。

Points 插件是会员专属应用，不随 SmartAdmin 开源版开放分发；仅会员授权环境可安装和使用。

Points 插件提供基于钉钉组织、钉钉加班审批和企业内部礼品库的积分兑换闭环。后台接口使用 System Token 和 `points.*` 权限码；员工端接口使用 `Plugin\Points\Model\PointsMember` Token，员工身份只能由钉钉免登换取，不接受前端直接提交钉钉 UserId 冒用。

## 后台钉钉配置与审批

| 方法 | 路径 | 权限码 | 说明 |
|---|---|---|---|
| GET | `/system/points/dingtalk/config` | `points.dingtalk.index` | 获取当前租户钉钉积分配置，Client ID、Client Secret 和回调 Token 只回显掩码。 |
| PUT | `/system/points/dingtalk/config` | `points.dingtalk.config` | 保存 CorpId、Client ID、Client Secret、AgentId、员工端地址、审批流程和字段映射。 |
| POST | `/system/points/dingtalk/sync` | `points.dingtalk.sync` | 同步钉钉部门员工到 Points 自有员工快照，不写入 System 用户。 |
| GET | `/system/points/dingtalk/sync-logs` | `points.dingtalk.log` | 查询组织同步日志。 |
| GET | `/system/points/dingtalk/user-binds` | `points.dingtalk.sync` | 查询钉钉员工绑定快照。 |
| GET | `/system/points/dingtalk/notify-logs` | `points.dingtalk.log` | 查询钉钉通知日志。 |
| GET | `/system/points/dingtalk/approvals` | `points.dingtalk.approval-import` | 查询加班审批导入记录。 |
| POST | `/system/points/dingtalk/approval-import` | `points.dingtalk.approval-import` | 按审批实例 ID 或时间范围补偿导入已通过加班审批。 |

## 后台积分、礼品与兑换

| 方法 | 路径 | 权限码 | 说明 |
|---|---|---|---|
| GET | `/system/points/rules/index` | `points.rule.index` | 查询加班审批积分规则。 |
| GET | `/system/points/rules/info/{id}` | `points.rule.index` | 获取积分规则详情。 |
| POST | `/system/points/rules/create` | `points.rule.create` | 新增加班审批积分规则。 |
| PUT | `/system/points/rules/update/{id}` | `points.rule.update` | 编辑积分规则。 |
| PUT | `/system/points/rules/status/{id}` | `points.rule.status` | 启用或禁用积分规则。 |
| DELETE | `/system/points/rules/delete/{ids}` | `points.rule.delete` | 删除积分规则。 |
| GET | `/system/points/members/index` | `points.member.index` | 查询员工积分账户和余额统计。 |
| GET | `/system/points/members/info/{id}` | `points.member.index` | 获取员工积分账户详情。 |
| PUT | `/system/points/members/status/{id}` | `points.member.status` | 启用或禁用员工积分账户。 |
| POST | `/system/points/members/adjust/{id}` | `points.member.adjust` | 手工调整员工积分，必须追加积分流水。 |
| GET | `/system/points/ledger/index` | `points.ledger.index` | 查询不可变积分流水。 |
| GET | `/system/points/gifts/index` | `points.gift.index` | 查询礼品列表。 |
| GET | `/system/points/gifts/info/{id}` | `points.gift.index` | 获取礼品详情。 |
| POST | `/system/points/gifts/create` | `points.gift.create` | 新增企业内部礼品。 |
| PUT | `/system/points/gifts/update/{id}` | `points.gift.update` | 编辑礼品积分价格、库存和说明。 |
| PUT | `/system/points/gifts/status/{id}` | `points.gift.status` | 上架或下架礼品。 |
| POST | `/system/points/gifts/stock/{id}` | `points.gift.stock` | 调整礼品库存。 |
| DELETE | `/system/points/gifts/delete/{ids}` | `points.gift.delete` | 删除礼品。 |
| GET | `/system/points/exchanges/index` | `points.exchange.index` | 查询兑换单。 |
| GET | `/system/points/exchanges/info/{id}` | `points.exchange.index` | 获取兑换详情。 |
| POST | `/system/points/exchanges/approve/{id}` | `points.exchange.audit` | 审核通过兑换。 |
| POST | `/system/points/exchanges/reject/{id}` | `points.exchange.audit` | 驳回兑换并退回积分和库存。 |
| POST | `/system/points/exchanges/deliver/{id}` | `points.exchange.deliver` | 确认礼品已发放，发放后不可撤销。 |
| GET | `/system/points/reports/overview` | `points.report.index` | 查询积分发放、消耗、余额和兑换概览。 |

## 员工端接口

| 方法 | 路径 | 认证 | 说明 |
|---|---|---|---|
| GET | `/points/account/auth/dingtalk-config` | 公开 | 获取钉钉免登所需 CorpId，不返回服务端密钥。 |
| POST | `/points/account/auth/dingtalk-login` | 公开 | 使用钉钉 auth code 登录积分员工端。 |
| POST | `/points/account/auth/refresh` | Points Token | 刷新员工端 Token。 |
| POST | `/points/account/auth/profile` | Points Token | 获取当前积分员工资料。 |
| GET | `/points/account/auth/codes` | Points Token | 获取员工端权限码。 |
| POST | `/points/account/auth/logout` | Points Token | 登出员工端。 |
| GET | `/points/me/balance` | Points Token | 获取我的积分余额、累计获得和累计消耗。 |
| GET | `/points/me/ledger` | Points Token | 查询我的积分流水。 |
| GET | `/points/gifts/index` | Points Token | 查询已上架礼品。 |
| GET | `/points/gifts/info/{id}` | Points Token | 获取已上架礼品详情。 |
| GET | `/points/exchanges/index` | Points Token | 查询我的兑换记录。 |
| POST | `/points/exchanges/create` | Points Token | 提交礼品兑换，立即扣减积分并占用库存。 |
| POST | `/points/exchanges/cancel/{id}` | Points Token | 取消待审核兑换并退回积分和库存。 |
| POST | `/points/dingtalk/approval-callback` | 回调 Token | 钉钉审批事件回调，校验配置 Token 后导入审批实例。 |

## 关键边界

- 加班积分只处理钉钉审批实例，不读取钉钉考勤自动计算加班。
- 积分余额只能通过积分流水变动；发放、扣减、退款都在事务内更新余额快照。
- 兑换提交时扣积分和占库存；驳回或取消退回；发放后不可撤销。
- 钉钉同步只维护 Points 自有员工快照，不自动创建 System 用户。
