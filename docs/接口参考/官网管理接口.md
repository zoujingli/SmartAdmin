# 官网管理接口

官网管理后台接口用于维护 Website 插件内部的站点、接口应用、栏目、导航、内容、页面区块和访客线索。本文档只覆盖后台 `/system/website/**` 管理接口，需要 System 登录态和对应权限码；面向官网前端或第三方服务端读取公开数据的签名接口，请查看 [官网开放接口](./官网开放接口.md)。

## 接口总览

后台菜单由 `plugin/Website/plugin.json` 提供，入口为 `/system/website`，包含“官网站点、接口应用、官网栏目、官网导航、官网内容、官网区块、访客线索”七个管理页。后台接口负责资料维护、发布状态和线索处理，不直接承载对外官网页面渲染。
